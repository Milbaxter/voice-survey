## HOW TO IDENTIFY IF THEY USE PHONE SURVEYS

### Ask These Questions on Discovery Calls:

1. **"Do you currently collect feedback from Swedish speakers in Finland?"**
   - If yes → "How do you do that?"

2. **"Do you use phone interviews, online surveys, or something else?"**
   - If phone → PERFECT FIT
   - If online only → Still fits (voice = richer data)

3. **"How much does a typical survey cost you?"**
   - If €8K-10K → They're using phone interviews or full-service firms

4. **"Who do you hire to do the surveys?"**
   - If Taloustutkimus/Norstat → They're outsourcing, expensive
   - If in-house → They have staff doing calls
